Explicação da matéria vista até agora

Resumo de cada coisa aprendida.

O que é html?

O html permite inserir o conteúdo e estabelecer a estrutura básica de um website. Ou seja é ele que da o significado e organizar as informações de uma página na web.

O que é Body?

O body é uma das tags mais importantes, pois é nela que fica todo o conteúdo visual que queremos exibir.

O que é o "p"?

O "p" serve para criarmos um paragrafo em nossos textos no Vs Code.

O que é "pre"?

A extensão do Visual Studio Code permite uma opção de pré-visualização em direto para ver a página de conteúdo de portais dentro do Visual Studio Code durante a experiência de desenvolvimento.

O que é "h1...h6"?

Esta tag representa o "Titulo" do artigo.

O que é span?

O span é um conteiner generico em linha para conteúdo fraseado , que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo.

O que é div?

O div é o que define uma divisão em um documento HTML.

O que é CSS?

O CSS é o código que você usa para dar estilo à sua página Web.

O que é Tag?

A tag é As tags são usadas para informar ao navegador a estrutura do site.

O que é classe?

A classe é uma forma de identificar os elementos HTML para serem modificados pelo CSS.

O que é id?

O id é um identificador único para modificar qualquer elemento, mas se ele for usado uma vez, não poderá ser usado novamente.

O que é Head?

O Head providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
